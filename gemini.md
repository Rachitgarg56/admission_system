# Project Context

This is an Admission Management System built with:
- Frontend: Next.js + Tailwind CSS
- Backend: Express.js (MVC)
- Database: MySQL

# Roles

- Admin: manages program and quota setup
- Admission Officer: manages applicants, seat allocation, confirmation
- Management: view dashboard only

# Important Business Rules

- No seat overbooking
- Seat is locked immediately after allocation
- Admission confirmed only after fee is paid
- Admission number is generated only once and is immutable

# Constraints

- Do NOT add authentication
- Do NOT change tech stack
- Do NOT overengineer
